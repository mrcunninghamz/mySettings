# mySettings

A centralized repository for storing operating system configurations and environment setups across different platforms.

## Purpose

This repository serves as a version-controlled backup and reference for:
- Configuration files (dotfiles)
- Environment setup scripts
- Tool-specific settings
- Terminal emulator configurations
- Window manager settings
- Shell configurations
- Utility preferences

## Structure

- **Mac/** - macOS configuration files and setup scripts
- **Windows/** - Windows configuration files and setup scripts
- **Linux/** - Linux distribution configuration files and setup scripts

## Status

⚠️ **Work in Progress** - These configurations are actively being developed and refined.

## Usage

See [claude.md](./claude.md) for the workflow to document and manage configurations.

## Contributing to Your Own Configurations

1. Navigate to your operating system folder
2. Add your configuration files
3. Document what each file does
4. Commit changes with descriptive messages

## Configuration Tracking

When adding new configurations, document:
- Operating system and version
- Tool/application name and version
- Purpose of the configuration
- Installation/setup instructions
- Dependencies (if any)
